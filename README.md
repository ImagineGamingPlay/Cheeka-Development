## ğŸ“‚ Setup

1. Clone this repository

    ```
    git clone https://github.com/ImagineGamingPlay/Cheeka-Development
    ```

2. Fill-in `.env.sample` file.
3. Rename `.env.sample` -> `.env`
4. Install all the dependencies

    ```python
    yarn
    ```

    OR

    ```
    node install
    ```

5. Build the project
    ```
    yarn build
    ```
    OR
    ```
    npm run build
    ```
6. Run your bot
    ```
    yarn start
    ```
    OR
    ```
    npm start
    ```
